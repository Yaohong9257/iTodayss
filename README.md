### 应用正常使用需要的权限

为了让计时提醒和番茄可以正常使用，需要开启以下权限：
* 1.在“权限管理”给应用添加”后台弹出界面”的权限，无此权限时，番茄倒计时结束无法弹框通知;
* 2.应用的“省电策略”—设置为“无限制”；
* 3.开启“自启动”；

注意：无2和3权限时，番茄倒计时服务会被系统kill，结束时无法通知，会出现负计时；
